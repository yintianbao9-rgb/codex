19 进阶与团队

排障手册

遇到问题不要先怀疑自己不会用，先按清单排查：账号、权限、路径、网络、依赖、模型、项目状态。

登录问题

  • 确认账号能否正常登录 ChatGPT。
  • 确认 App / CLI / IDE 是不是同一个账号。
  • API Key 和 ChatGPT 登录不要混淆。

文件问题

  • 确认选对项目文件夹。
  • 确认文件没有被系统权限拦住。
  • 确认 AI 是否只读不写。

网页问题

  • 本地服务是否启动。
  • 端口是否被占用。
  • 图片路径是否 200。
  • 浏览器是否缓存旧文件。

任务质量问题

如果输出空泛，通常是资料不足、目标不清、边界没写、验收标准缺失。回到任务模板重写。
